## 1. Introduction
This project is a RESTful backend API for a social networking application.
It provides core social features such as authentication, user profiles, posts,
and real-time messaging between users.

---

## 2. Tech Stack
- ASP.NET Core Web API  
- Entity Framework Core  
- JWT Authentication  
- SignalR (Real-time communication)  
- SQL Server / Cassandra
- Clean Architecture (3 Layers)

---

## 3. Core Features
- User registration and authentication using JWT
- Login with Goole account
- User profile management
- Create, update, and delete posts
- Like and comment on posts
- Follow and unfollow users
- Basic news feed and suggest friend logic
- One-to-one real-time chat using SignalR
- Manage group
